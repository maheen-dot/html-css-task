# EduSphere - Responsive Online Learning Platform

## Overview

EduSphere is a responsive landing page for an online learning platform built using HTML5 and CSS3. The project demonstrates modern web layout techniques including Flexbox, CSS Grid, responsive design, the CSS box model, positioning, and media queries.

This project was developed as part of a frontend web development assignment to practice creating a multi-section static webpage.

---

## Features

* Responsive navigation bar with hamburger menu
* Hero section with call-to-action
* Popular course categories
* Featured courses section
* Why Choose EduSphere section
* Student testimonials
* Responsive footer
* Mobile-first responsive design

---

## Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Grid
* Media Queries
* Font Awesome Icons
* Google Fonts (Poppins)

---

## Concepts Demonstrated

This project includes the following HTML and CSS concepts:

* Semantic HTML elements (nav, section, footer)
* Flexbox layouts
* CSS Grid layouts
* Responsive web design
* CSS Box Model
* Margin and Padding
* Relative and Absolute Positioning
* Image responsiveness

## Responsive Design

The website is fully responsive and optimized for:

* Desktop (1200px and above)
* Tablet (992px and below)
* Mobile devices (768px and below)

Responsive features include:

* Hamburger navigation menu
* Responsive course cards
* Adaptive Grid and Flexbox layouts
* Scalable images
* Mobile-friendly spacing and typography
